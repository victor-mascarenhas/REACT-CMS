# CMS - Chido Tacos

Um portal para restaurantes com gerenciador de produtos feito em React.

Portal View:
https://chido-tacos.vercel.app

Admin View:
https://chido-tacos.vercel.app/admin


Bibliotecas utilizadas:

    axios
    bootstrap
    node-sass
    nodemailer-react
    react
    react-bootstrap
    react-icons
    react-intl-currency-input
    react-router-dom    
    styled-components
    sweetalert2


## Instalação

```bash
npm install
```
